# Fitness Wearable Engagement Analysis (FitSync Case Study)
*Analysis conducted for FitSync, a fictional fitness wearable company*
Analyzing fitness wearable user behavior to identify key drivers of engagement and provide data-driven recommendations for increasing high-value users.

## 📊 Overview
This project analyzes user behavior data from a fitness wearable company (FitSync) to identify the key drivers of user engagement and provide actionable business recommendations.

## 🎯 Business Problem
FitSync aims to increase user engagement and retention. This analysis explores which user behaviors are most strongly associated with high engagement levels.

## 🧠 Key Insight
High engagement is primarily driven by high-intensity activity (very active minutes), rather than gradual increases in lower-intensity movement.

## 📈 Analysis Highlights
- Segmented users into Low, Medium, and High engagement tiers
- Developed a weighted engagement score based on activity intensity
- Identified significant differences in high-intensity activity across user groups

## 🚀 Recommendations
1. Implement gamified challenges and leaderboards to encourage high-intensity activity
2. Introduce progressive intensity programs to help users increase engagement over time

## 🛠 Tools Used
- Google Sheets (data cleaning, feature engineering, analysis)

## 📊 Key Metric
Engagement Score:
(Very Active * 3) + (Fairly Active * 2) + (Lightly Active * 1)
